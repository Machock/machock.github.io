字符串和数字之间的比较

规范定义:

(1) 如果Type(x)为数字,Type(y)是字符串,则返回 x == ToNumber(y)
(2) 如果Type(x)为字符串,Type(y)是数字,则返回 ToNumber(x) == y
结论将字符串执行ToNumber() 后进行比较

字符串和数字之间的比较

规范定义:

(1) 如果Type(x)为布尔类型,则返回 ToNumber(x) == y
(2) 如果Type(y)为布尔类型,则返回 x == ToNumber(y)
结论将布尔值执行ToNumber() 后进行比较

null 和 undefined 之间的比较

规范定义:

(1) 如果x为null,y为undefined,返回true
(2) 如果x为undefined,y为null,返回true
结论: 使用null == undefined 是安全可靠的,避免了0,false和’’这样的假值

对象 和 非对象 之间的比较

规范定义:

(1) 如果Type(x)为字符串或数字,Type(y)为对象,则返回 x == ToPrimitive(y)
(2) 如果Type(x)为对象,Type(y)为字符串或数字,则返回 ToPrimitive(x) == y

null 和undefined 均不能被封装,Object(null) 得到 一个常规对象 {}
NaN能够被封装为数字封装对象,拆封后 NAN == NAN 为 false
